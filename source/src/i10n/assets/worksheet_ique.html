Dr.Mario Asset Localization Worksheet (汉化制作表)

Message Text Translation

  • input: assets/mes/*.mes
  • output: assets/mes/*_zh.mes
  • notes:
    • translate the text in between double quotes, preserve the rest of the texts
    • preserve all special characters preceded with "~"
    • preserve letters A, B, C, Z, L, R

Character Font Glyph

  • input:
    • assets/mes/*_zh.mes
  • output:
    • assets/font/font_c.bmp, assets/font/font_c2.bmp (12x12 font glyph bmp)
  • tools: Photoshop, Yongzhi's character generation tools
  • notes:
    • use assets/font/font_c.bmp & assets/font/font_c2.bmp as example
    • for each unique Chinese character used in the *_zh.mes files, generate a font glyph at size 12x12
    • the output bmp files should be indexed 8bit bmp, use the same color index table as the ones in assets/font/font_c.bmp & assets/font/font_c2.bmp

Texture

  • input:
    • assets/texture/*/*.bmp
  • output:
    • assets/texture_zh/*/*.bmp
  • tools: Photoshop
  • notes:
    • translate the bmp files that have English text in them, generate the corresponding Chinese texture bmp files
    • keep color schemes consistent
    • keep the original dimension of the images
